# Multilingual Polarization Detection

## Description
This project detects polarization in multilingual text using machine learning.

## Features
- TF-IDF Vectorization
- Logistic Regression
- Flask Web App

## How to Run
1. Activate environment
2. Run: python app/app.py
3. Open browser: http://127.0.0.1:5000
